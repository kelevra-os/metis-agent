# metis-wormhole-intake

Cross-device file intake via Wormhole for Metis.

## Status

🧱 Scaffold — coming in a future phase.

## Planned Capabilities

- Receive files from other devices via Wormhole protocol
- Link-based file intake (PrivateBin, sharing links)
- Automatic file routing to Obsidian vault attachments
- File type detection and processing
- Large file chunking and reassembly

## Configuration

Optionally requires `WORMHOLE_API_KEY` in the Metis `.env`.
